## Features
アサルトリリィに登場するルーン文字をアルファベットを相互変換するプログラムをつくりました。
アルファベット以外の文字は変換されずにそのままになるようになっています。

## How to use
リリースのzipファイルをダウンロードして解答し、中にあるRuneConverter.exeを実行してください。

## Develop Enviroment
Python 3.9.6  
PySimpleGUI 4.46.0  
PyInstaller 4.5.1  
※実行には必要ありません。
